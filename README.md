# GovukPersonalisation

A gem to hold shared code which other GOV.UK apps will use to implement
accounts-related functionality.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'govuk_personalisation'
```

And then execute:

```
$ bundle install
```
